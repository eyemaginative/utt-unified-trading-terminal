# Frequently Asked Questions (FAQ)

## Is UTT open source right now?

The project can be developed in a private GitHub repo first, then made public when core security and docs are ready.

## Can I run UTT without real funds?

Yes—paper/simulated trading support is recommended before connecting any live account.

## Does UTT store exchange/broker API keys?

Keys should be kept in environment variables or secret storage, never hardcoded in source control.

## Is UTT financial advice?

No. UTT is a software platform for trading operations and research.

## How do I report bugs?

Open a GitHub issue with steps to reproduce, expected behavior, and logs/screenshots.

## How do I report security issues?

Follow `SECURITY.md` and report privately.

## When should the repo go public?

After removing secrets, validating docs, and setting branch protection and security controls.
