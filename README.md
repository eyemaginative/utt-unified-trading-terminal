# UTT (Unified Trading Terminal)

UTT is your trading project, now prepared to be moved into a private GitHub repository and gradually opened to contributors.

> This repository currently focuses on **project governance and onboarding docs** so you can confidently publish and maintain it as open source later.

## What this repo includes

- A step-by-step GitHub migration checklist for beginners (`docs/GITHUB_MIGRATION.md`)
- Security policy and vulnerability reporting flow (`SECURITY.md`)
- Whitepaper template/outline to explain UTT's vision (`docs/WHITEPAPER.md`)
- FAQ for common community questions (`docs/FAQ.md`)
- User help and troubleshooting guide (`docs/HELP.md`)
- Build and usage guide template (`docs/BUILD_AND_USE.md`)
- Contributing guide for future collaborators (`CONTRIBUTING.md`)

## Suggested rollout path

1. Move code to private GitHub repo using the migration guide.
2. Add actual source code and dependencies.
3. Fill in build/run commands in `docs/BUILD_AND_USE.md`.
4. Publish initial whitepaper draft.
5. Open private beta with trusted testers.
6. Later switch repo visibility to public when ready.

## Beginner-friendly note

If you are new to GitHub: follow `docs/GITHUB_MIGRATION.md` exactly in order. It includes copy/paste commands and verification steps.
